# Slots are **Bytes**

- A **bit** is just a $0$ or $1$. 
- A **byte** is a collection of $8$ bits. 
- A "slot" in memory is a **byte** (usually).


A laptop with $16$ GB of RAM has roughly 

- **1.6 billion** bytes (= "slots" in memory), or
- **12.8 billion** zeros and ones.
